# -> Forwarder
>You own your Data

Forward your Octoblu data to data stores such as
- Splunk
- Elastic
- Microsoft
- Azure
- MongoDB
- TempoIQ

## Octoblu Dev set-up/install
1. Go to octoblu-dev
1. Run the following stacks:
  1. meshblu
  1. octoblu
  1. forwarder
1. Go to: [https://forwarder.octoblu.dev](https://forwarder.octoblu.dev)
